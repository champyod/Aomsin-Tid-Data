# Sample Data

This directory contains sample datasets for analysis.

## Directory Structure

- `raw/`: Place your raw, unprocessed data files here
- `processed/`: Processed and cleaned data will be saved here

## Supported File Formats

- CSV (`.csv`)
- Excel (`.xlsx`, `.xls`)
- JSON (`.json`)
- Parquet (`.parquet`)

## Example Data Files

You can download sample datasets from:
- [Kaggle](https://www.kaggle.com/datasets)
- [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php)
- [Data.gov](https://www.data.gov/)
- [Google Dataset Search](https://datasetsearch.research.google.com/)

## Tips

- Keep your raw data unchanged
- Save processed data to the `processed/` directory
- Use descriptive filenames
- Document your data sources in your notebooks
